# 샘플 파일 형식 가이드

⚠️ **주의: 이 샘플 파일들은 모두 더미 데이터입니다. 실제 학생 정보가 아닙니다.**

---

## 📋 **필수 파일**

### 1. **학생명 파일** (sample_students.csv) ✨ 필수
- **형식**: `수험번호,이름` 또는 `수험번호,성명`
- **용도**: 수험번호를 학생 이름으로 매핑
- **💡 Tip**: "이름" 또는 "성명" 둘 다 인식합니다!

```csv
수험번호,이름
2024001,김철수
2024002,이영희
2024003,박민수
```

또는

```csv
수험번호,성명
2024001,김철수
2024002,이영희
2024003,박민수
```

### 2. **과목 성적 파일** (sample_korean.csv, sample_math.csv 등) ✨ 필수
- **형식**: `수험번호,과목코드,총점,만점,정답수,오답번호`
- **용도**: 학생별 과목 성적

---

## 📊 등급컷 파일 (sample_grade_cutoff.csv) ⭐ 선택사항

**등급컷 파일은 선택사항입니다!** 없어도 성적표를 생성할 수 있습니다.

### 필수 컬럼 (등급컷 파일을 사용할 경우)
- `과목명`: 과목 이름
- `과목코드`: 과목 식별 코드

### 선택 컬럼
- `1등급컷` ~ `9등급컷`
- `만점표점`, `1등급표점` ~ `9등급표점`
- `1등급백분위` ~ `9등급백분위`

## 📄 **통일된 과목 파일 형식**

### **모든 과목이 동일한 형식!**

```csv
수험번호,과목코드,총점,만점,정답수,오답번호
```

### 예시 - 국어 (sample_korean.csv)
```csv
수험번호,과목코드,총점,만점,정답수,오답번호
2024001,2,100,100,45/45,없음
2024002,2,97,100,44/45,29
2024003,1,95,100,43/45,"24, 29"
```

### 예시 - 수학 (sample_math.csv)
```csv
수험번호,과목코드,총점,만점,정답수,오답번호
2024001,1,100,100,30/30,없음
2024002,2,96,100,28/30,"20, 30"
2024003,1,92,100,27/30,"15, 22, 29"
```

### ⭐ **탐구 파일 1개로 통합!** (sample_inquiry.csv)

**같은 수험번호가 2줄씩!** 탐구 1과 탐구 2를 하나의 파일에 입력하세요.

```csv
수험번호,과목코드,총점,만점,정답수,오답번호
2024001,11,50,50,20/20,없음
2024001,20,50,50,20/20,없음
2024002,12,49,50,19/20,12
2024002,21,49,50,19/20,18
2024003,13,48,50,19/20,8
2024003,22,48,50,19/20,12
```

**자동으로 과목코드 순서대로 배치됩니다!**
- 첫 번째 과목코드 → 탐구1
- 두 번째 과목코드 → 탐구2

## 🎯 **사용 팁**

### 1. 파일 업로드 순서
```
1️⃣ 학생명 파일 (필수)
2️⃣ 등급컷 파일 (선택)
3️⃣ 국어, 수학, 영어, 한국사, 탐구 파일
```

### 2. 탐구 파일 작성 방법
```
✅ 좋은 예: sample_inquiry.csv 1개에 모든 학생의 탐구1, 탐구2
❌ 나쁜 예: sample_inquiry1.csv, sample_inquiry2.csv 2개로 분리
```

### 3. 같은 수험번호 2줄 작성
```csv
수험번호,과목코드,총점,만점,정답수,오답번호
2024001,11,50,50,20/20,없음        ← 탐구1
2024001,20,50,50,20/20,없음        ← 탐구2 (같은 수험번호)
```

### 4. 과목코드 자동 정렬
- 시스템이 과목코드 순서대로 자동 정렬
- 작은 코드 → 탐구1
- 큰 코드 → 탐구2
