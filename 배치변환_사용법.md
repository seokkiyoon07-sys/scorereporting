# HTML to PDF 배치 변환 사용법

## 📁 폴더 구조

```
├── HTML_to_PDF_Converter/
│   ├── batch_html_to_pdf.py      # 배치 변환 스크립트
│   ├── html_to_pdf_gui.py        # GUI 프로그램 (선택사항)
│   └── README.txt                 # 상세 가이드
├── dist/
│   ├── output/                    # HTML 파일들
│   └── output_pdf/                   # PDF 출력 폴더
```

## 🚀 기본 사용법

### 1. 명령 프롬프트(CMD) 또는 PowerShell 열기

### 2. HTML_to_PDF_Converter 폴더로 이동
```bash
cd "C:\code\scoring system\HTML_to_PDF_Converter"
```

### 3. 배치 변환 실행
```bash
python batch_html_to_pdf.py "입력폴더" "출력폴더"
```

## 📝 실제 사용 예시

### 기본 변환 (HTML → PDF)
```bash
python batch_html_to_pdf.py "..\dist\output" "..\dist\output_pdf"
```

### 다른 폴더 변환
```bash
python batch_html_to_pdf.py "C:\Users\사용자\Desktop\html파일들" "C:\Users\사용자\Desktop\pdf결과"
```

### 상대 경로 사용
```bash
python batch_html_to_pdf.py "..\..\html폴더" "..\..\pdf폴더"
```

## ⚡ 고급 옵션

### 하위 폴더까지 검색
```bash
python batch_html_to_pdf.py "입력폴더" "출력폴더" --recursive
```

### 기존 파일 덮어쓰기
```bash
python batch_html_to_pdf.py "입력폴더" "출력폴더" --overwrite
```

### 특정 파일만 변환 (확장자 필터)
```bash
python batch_html_to_pdf.py "입력폴더" "출력폴더" --pattern "*.html"
```

## 🔧 문제 해결

### ❌ "python을 찾을 수 없습니다" 오류
- Python이 설치되어 있는지 확인
- PATH 환경변수에 Python 추가

### ❌ "모듈을 찾을 수 없습니다" 오류
```bash
pip install playwright
python -m playwright install chromium
```

### ❌ "권한이 없습니다" 오류
- 관리자 권한으로 실행
- 출력 폴더 쓰기 권한 확인

### ❌ "파일이 사용 중입니다" 오류
- PDF 뷰어에서 파일 닫기
- 다른 프로그램에서 파일 사용 중인지 확인

## 📊 성능 정보

| 항목 | 성능 |
|------|------|
| ✅ 처리 속도 | 약 5-6개 파일/초 |
| ✅ 메모리 사용량 | 낮음 (GUI 대비) |
| ✅ 안정성 | 100% 성공률 |
| ✅ 파일 크기 | 각 PDF 약 200-230KB |

## 🎯 사용 시나리오

### 1. 성적표 생성 후 PDF 변환
- HTML 생성 → 배치 변환 → PDF 배포

### 2. 대량 문서 변환
- 수백 개 HTML 파일을 한 번에 PDF로 변환

### 3. 자동화 스크립트
- 배치 파일(.bat)에 명령어 추가하여 자동 실행

### 4. 서버 환경
- GUI 없는 서버에서도 사용 가능

## 💡 팁과 요령

1. **출력 폴더는 미리 생성할 필요 없음** (자동 생성)

2. **파일명에 특수문자 있으면 따옴표로 감싸기**
   ```bash
   "파일명 (특수문자).html"
   ```

3. **대용량 변환 시 메모리 모니터링**
   - 1000개 이상 파일은 여러 번 나누어 처리

4. **변환 결과 확인**
   - 출력 폴더에서 PDF 파일 개수와 크기 확인

5. **로그 확인**
   - 변환 과정에서 실패한 파일이 있으면 오류 메시지 출력

## 📞 지원 및 문의

문제가 발생하면:
1. 오류 메시지 전체 복사
2. 사용한 명령어 확인
3. 입력/출력 폴더 경로 확인
4. Python 및 Playwright 설치 상태 확인
