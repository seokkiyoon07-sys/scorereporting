HTML → PDF 배치 변환 도구 사용 가이드
========================================

📋 개요
- HTML 파일을 한번에 PDF로 변환하는 배치 처리 도구
- Playwright 기반 고품질 PDF 생성
- 폴더 내 모든 HTML 파일 일괄 처리

🚀 사용 방법

1. 단일 파일 변환:
   python batch_html_to_pdf.py single_file.html

2. 폴더 내 모든 HTML 파일 변환:
   python batch_html_to_pdf.py ./html_files

3. 출력 폴더 지정:
   python batch_html_to_pdf.py ./html_files ./pdf_output

4. 하위 폴더까지 재귀 검색:
   python batch_html_to_pdf.py ./html_files ./pdf_output -r

5. 기존 PDF 파일 덮어쓰기:
   python batch_html_to_pdf.py ./html_files ./pdf_output -o

6. 모든 옵션 사용:
   python batch_html_to_pdf.py ./html_files ./pdf_output -r -o

📊 주요 기능

✅ 배치 처리
- 폴더 내 모든 HTML 파일 자동 검색
- 하위 폴더 재귀 검색 지원
- 폴더 구조 유지

✅ 고품질 PDF 생성
- Playwright Chromium 엔진 사용
- 한글 폰트 완벽 지원
- CSS 스타일 보존
- A4 페이지 최적화

✅ 사용자 친화적
- 진행 상황 실시간 표시
- 상세한 결과 요약
- 오류 처리 및 로깅

⚠️ 시스템 요구사항

- Python 3.7 이상
- Playwright 설치 필요:
  pip install playwright
  python -m playwright install chromium

- Windows 10/11 권장
- 최소 4GB RAM

📁 파일 구조 예시

입력 폴더:
html_files/
├── report1.html
├── report2.html
└── subfolder/
    └── report3.html

출력 폴더 (재귀 모드):
pdf_output/
├── report1.pdf
├── report2.pdf
└── subfolder/
    └── report3.pdf

🔧 고급 사용법

1. 프로그래밍 방식 사용:
   from batch_html_to_pdf import batch_html_to_pdf, convert_single_html
   
   # 단일 파일 변환
   convert_single_html("input.html", "output.pdf")
   
   # 배치 변환
   batch_html_to_pdf("./html_files", "./pdf_output", recursive=True)

2. 커스텀 설정:
   - HTML 파일 인코딩: UTF-8 권장
   - CSS 미디어 쿼리: @media print 지원
   - 폰트: 시스템 폰트 자동 감지
