SN독학기숙학원 성적 관리 시스템 - CSV 템플릿 가이드
====================================================

📋 제공된 CSV 템플릿 파일들:

1. grade_cutoff_template.csv
   - 등급컷 데이터 템플릿
   - 각 과목별 1-9등급 컷오프 점수
   - 최대 표준점수 포함

2. grade_standard_scores_template.csv  
   - 등급별 표준점수 템플릿
   - 각 과목별 1-9등급 표준점수
   - 영어/한국사는 표준점수 없음 (0)

3. subject_codes_template.csv
   - 과목 코드 매핑 테이블
   - 과목명, 과목코드, 카테고리 정보

4. student_data_template.csv
   - 학생 성적 데이터 템플릿
   - 이름, 과목명, 과목코드, 총점, 정답수, 오답번호

📊 CSV 파일 사용 방법:

1. 등급컷 및 표점 설정:
   - grade_cutoff_template.csv 수정
   - 각 과목별 등급 컷오프 점수 입력
   - Max_Standard_Score 컬럼에 만점 표준점수 입력

2. 학생 데이터 입력:
   - student_data_template.csv 참고
   - 학생별 성적 데이터 입력
   - 오답번호는 쉼표로 구분하여 입력

3. 프로그램에서 사용:
   - GUI의 "등급컷 CSV 업로드" 버튼 사용
   - 또는 직접 GUI에서 등급컷/표점 입력

⚠️ 주의사항:
- CSV 파일은 UTF-8 인코딩으로 저장
- 영어 과목명 사용 권장 (한글명도 지원)
- 오답번호는 숫자와 쉼표로만 구성
- 빈 값은 0으로 처리됨

🔧 데이터 형식:
- Total_Score: 0-100 (또는 0-50 for 탐구과목)
- Correct_Count: 정답 개수
- Wrong_Answers: 오답번호 (예: "3,7,15,22")
- Subject_Code: 2자리 숫자 (예: "01", "05")
