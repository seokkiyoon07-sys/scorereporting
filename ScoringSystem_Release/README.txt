SN독학기숙학원 성적 관리 시스템 v2.0
========================================

📋 프로그램 개요
- 학생 성적 데이터 처리 및 PDF 성적표 생성
- 등급컷 및 표점 데이터 직접 입력 지원
- Playwright 기반 고품질 PDF 생성

🚀 사용 방법
1. ScoringSystem.exe 실행
2. 과목별 CSV/Excel 파일 업로드
3. 등급컷 및 표점 데이터 입력 후 저장
4. 데이터 처리 버튼 클릭
5. PDF 생성 버튼 클릭

📁 샘플 파일
- sample_korean.csv: 국어 샘플 데이터
- sample_math.csv: 수학 샘플 데이터  
- sample_english.csv: 영어 샘플 데이터
- sample_history.csv: 한국사 샘플 데이터
- sample_inquiry1.csv: 탐구1 샘플 데이터
- sample_inquiry2.csv: 탐구2 샘플 데이터

📁 CSV 템플릿 파일
- grade_cutoff_template.csv: 등급컷 데이터 템플릿
- grade_standard_scores_template.csv: 등급별 표점 템플릿
- subject_codes_template.csv: 과목 코드 매핑
- student_data_template.csv: 학생 데이터 템플릿
- CSV_Templates_Guide.txt: CSV 템플릿 사용 가이드

🔧 주요 기능
- GUI 기반 직관적 인터페이스
- 등급컷 및 표점 데이터 직접 입력
- 과목별 파일 삭제 기능
- 스크롤 가능한 인터페이스
- 자동 PDF 생성 (Playwright 기반)
- 한글 폰트 완벽 지원

📊 지원 파일 형식
- CSV 파일 (.csv)
- Excel 파일 (.xlsx)

⚠️ 주의사항
- Windows 10/11 환경에서 테스트됨
- 최소 4GB RAM 권장
- 인터넷 연결 필요 (Playwright 브라우저 다운로드)
